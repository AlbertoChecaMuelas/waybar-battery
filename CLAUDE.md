# waybar-battery

Scripts and configuration for showing peripheral battery levels in Waybar on Hyprland/Omarchy (CachyOS).

## Scope

- `razer-battery.py` — Waybar exec script that queries the Razer Naga V2 Pro battery via OpenRazer D-Bus and emits JSON for a `custom/mouse-battery` Waybar module.

## Deployment

Scripts are deployed to `~/.config/waybar/scripts/`. Waybar config and style live in `~/.config/waybar/`.

## Language

Always write in English.
